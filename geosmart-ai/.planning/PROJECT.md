# PROJECT: GeoSmart AI - Government-Level Next.js Migration

## Vision
Transform the existing GeoSmart AI React/Vite prototype into a production-grade, highly scalable "Government-Level" crisis management platform using **Next.js** for the frontend and a robust **Express** backend.

## Key Features to Preserve & Enhance
1. **Multilingual Intelligence**: Gemini Pro integration for 10+ Indian languages.
2. **Early Warning System**: Real-time hazard prediction ticker.
3. **Inventory Management**: Automated supply chain tracking.
4. **Crowdsourced Verification**: Anti-spam logic via GPS and reporting phone analysis.
5. **Role-Based Access Control (RBAC)**: Secure Official/Volunteer portal.

## Technical Goals (Milestone 1)
- [ ] Migrate Frontend to **Next.js (App Router)**.
- [ ] Implement **Server-Side Rendering (SSR)** for critical dashboards.
- [ ] Setup **Next.js API Routes** for lightweight orchestration.
- [ ] Maintain the existing **Express/Mongoose** core for mission logic.
- [ ] Ensure **100% Visual Fidelity** and "Wow" aesthetics.

## Roadmap
- **Phase 1**: Next.js Setup & Layout Architecture.
- **Phase 2**: Porting Core Components (Map, Bot, Dashboard).
- **Phase 3**: Authentication & RBAC Integration.
- **Phase 4**: Verification & Stress Testing.
